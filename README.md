# Whimbrel Habitat Selection Analysis

R code supporting the MSc dissertation:

"Soil Moisture and Prey Accessibility as Drivers of Whimbrel (Numenius phaeopus) Stopover Habitat Selection in the Lower Derwent Valley"

The script includes:
- Data processing
- Calculation of field-level means
- Spearman's rank correlations
- Linear regressions
- Wilcoxon rank-sum tests
- Figure generation

Institution: Cornwall College University Centre
